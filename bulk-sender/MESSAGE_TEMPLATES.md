# Message Templates

## Template 1: General Promotion
```
Hello {name}!

We have an exciting offer for you in {city}.

Valid until {date} at {time}.

Reply STOP to unsubscribe.
```

## Template 2: Event Invitation
```
Hi {name},

You're invited to our exclusive event in {city} on {date}.

RSVP by replying to this message.

See you there!
```

## Template 3: Product Launch
```
Dear {name},

We're launching a new product on {day}, {date}!

As a valued {tag}, you get early access.

Check it out now!
```

## Template 4: Reminder
```
Hi {name},

This is a friendly reminder about your appointment in {city}.

Date: {date}
Time: {time}

Reply STOP to opt out.
```

## Template 5: Survey Request
```
Hello {name}!

We'd love your feedback! As a {tag} from {city}, your opinion matters.

Please take 2 minutes to complete our survey.

Thank you!
```

## Variables Available:
- {name} - Contact's name
- {city} - Contact's city
- {tag} - Contact's tag/category
- {date} - Current date (e.g., "April 25th 2026")
- {time} - Current time (e.g., "10:30 AM")
- {day} - Current day (e.g., "Friday")
