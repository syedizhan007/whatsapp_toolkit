┌────────────────────────────────────────────────────────────┐

│   GROQ_API_KEY: 'your-key-here'                           │

┌────────────────────────────────────────────────────────────┐

└────────────────────────────────────────────────────────────┘

Step 4: Run Auto-Reply Bot
┌────────────────────────────────────────────────────────────┐
│   npm run auto-reply                                       │
└────────────────────────────────────────────────────────────┘

Step 5: Scan QR Code
┌────────────────────────────────────────────────────────────┐
│ - QR code appears in terminal                              │
