## 🚀 HOW TO USE

### Step 1: Open Website
```
http://localhost:3000/#
```

### Step 2: Go to Bulk Sender
- Click "Bulk Sender" in left sidebar

### Step 3: Create Campaign
1. **Campaign Name:** Enter name
2. **Message:** Write message (use `{name}` for personalization)
3. **Media (Optional):** Attach images/PDFs
4. **CSV File:** Upload contacts
   ```csv
   name,phone
   Ali Khan,923001234567
   Sara Ahmed,923217654321
   ```
5. **Click:** "Create Campaign"

### Step 4: Start Campaign
- Find campaign in table
- Click play button (▶️)
- Watch real-time progress
